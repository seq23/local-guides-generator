## GitHub Rules

- Owner is the only person who pushes to `main`
- VAs do NOT push code
- Repo is treated as production infrastructure

### Granting GitHub Access
Recommended:
- Read-only access for all VAs
- No write access unless absolutely required

Never:
- allow force-push
- share credentials
