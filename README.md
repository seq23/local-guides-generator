# Sponsors / Ads / Buyouts Runbook Bundle

Drop these folders into your repo:
- `docs/runbooks/sponsors_ads_buyouts/`
- `scripts/sponsors/`
- `data/sponsor_intake/` (examples + templates)

Then add the package.json scripts snippet from:
- `docs/runbooks/sponsors_ads_buyouts/07_package_json_snippet.md`

Run:
- `npm run sponsors:apply`
- `npm run sponsors:verify`

Optional Cloudflare Images mode:
- `npm run sponsors:apply:cf`
